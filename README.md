# Animated Background (React Three Fiber)

A premium, interactive **Obsidian Architecture** scrollytelling background built with React, Three.js, and React Three Fiber. 

This project perfectly mimics the high-end, full-screen cinematic 3D scrolling experiences seen on top-tier design agency websites (like Ascend Marketing or Apple product pages).

## How to Run Locally

Follow these exact steps in your command prompt to get the project ready and running:

1. **Install Dependencies**
   Run the following command to download and install all required packages (including React, Three.js, and `@react-three/drei`):
   ```bash
   npm install
   ```

2. **Start the Development Server**
   Once the installation is complete, start the local development server by running:
   ```bash
   npm run dev
   ```

3. **View the Website**
   Open your browser and navigate to the local URL provided in your terminal (usually `http://localhost:5173/` or `http://localhost:5175/`).

## Features
- **Cinematic Scrollytelling**: Your scroll wheel acts as a camera director, flying you at warp speed down an infinite 3D corridor.
- **Premium Materials**: Utilizes dark, highly reflective Obsidian/Glass textures combined with sharp studio rim lighting and environmental reflections.
- **Dynamic CSS Integration**: A rich radial CSS gradient perfectly tracks your mouse cursor underneath the 3D scene.
- **Industry Standard 3D**: Uses `@react-three/fiber` and `@react-three/drei` for rock-solid performance and perfect animation-loop handling.
